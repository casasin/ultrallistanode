# README

Ultrallista d'events musicals (en aquest cas prop de Santa Coloma de Queralt) fàcilment personalitzable.

## Thanks

Utilitza [opensheet](https://github.com/benborgers/opensheet#readme) per generar un `.json` a partir d'una *Google Spreadsheet* que conté les dades dels concerts. S'ha d'ordenar per data en aquest arxiu d'entrada de les dades.
